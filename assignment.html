meaning of tags

An HTML tag is a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document. As part of an HTML element, HTML tags help web browsers convert HTML documents into web pages.

An HTML tag is commonly defined as a set of characters constituting a formatted command for a Web page.



commonly used tags

  1. HTML TAG(html> Statements... /html>)
  2. HEAD TAG(head> Statements... /head>)
  3. BODY TAG(body> Statements... /body>)
  4. TITLE TAG(title> Statements... /title>)
  5. Heading tag(h1,h2,h3,h4,h5,h6)
  6. paragraph tag(p>........../p>)
  7. Font Style tag(em,b,small,i,u,strike,etc)

meaning of Element.

An HTML element is defined by a start tag, some content, and an end tag. In html every tagname that has a start tag and an end tag is an element.

Example

start tag element content end tag
h1 My First Heading /h1
p my first paragraph /p
br none none

meaning of Value.

The value attribute specifies the value of an "input" element. The value attribute is used differently for different input types: For "button", "reset", and "submit" - it defines the text on the button. For "text", "password", and "hidden" - it defines the initial (default) value of the input field.



meaning of Attribute.

HTML attributes are unique phrases which give additional data about the factors or attributes are the modifier of the HTML element.



examples

  • The href attribute of "a" specifies the URL of the page the link goes to·
  • The src attribute of :img: specifies the path where the image is located.
